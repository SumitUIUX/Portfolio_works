UI/UX Case Study

Revolution Travel CRM

Category
UI/UX Design
Project Duration
2 Months
Start Date
2 Jaunary 2023
Designer
Sumit Barapatre

challenges

From a design perspective, creating a comprehensive travel planner with integrated CRM involves managing diverse data types efficiently within a database architecture that supports fast and easy retrieval for report generation. The challenge lies in seamlessly incorporating inbuilt email management (e.g., Gmail) alongside integrations with instant messaging platforms (Instagram, Facebook, Twitter), ensuring a cohesive user experience. Additionally, the design must focus on crafting an intuitive, drag-and-drop campaign builder, facilitating a no-code environment for users to easily create and customize their own websites.

solutions

From a UX standpoint, the solution for advanced report generation was structured by pre-mapping key parameters before development, enabling efficient database utilization. For real-time instant messaging and email management, we implemented a process that runs every 30 seconds, designed to minimize server load while maintaining real-time responsiveness. The code architecture was optimized through the use of SDKs, and a cron job was scheduled to run every minute. The campaign builder was designed with distinct stages, connecting nodes between start and endpoints seamlessly. Additionally, real-time auto-save functionality was incorporated to ensure no data loss or rework for travel agents.
